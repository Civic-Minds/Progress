# Transit Construction | North America

An interactive Mapbox map of major transit construction projects across North America, styled like a magazine spread.

## Run locally

```bash
npm start
```

You will need a Mapbox access token to launch the map.

## Data

- Project data lives in `data/projects.json`.
- Entries include real source links and a mix of line and point geometries.

## Version

Current version: `1.2.0`
